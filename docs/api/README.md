# x0tta6bl4 API Документация

## Обзор

x0tta6bl4 предоставляет полный набор API-интерфейсов для работы с квантовыми системами, аутентификацией, платежами и мониторингом. Все API построены на принципах φ-гармонии (φ = 1.618) и используют современные стандарты безопасности.

## Базовая информация

- **Базовый URL**: `https://api.x0tta6bl4.com`
- **Версия API**: v1
- **Формат ответа**: JSON
- **Аутентификация**: Bearer токены (JWT)
- **Кодировка**: UTF-8

## Аутентификация

Большинство API требуют аутентификации через Bearer токен:

```
Authorization: Bearer <your_jwt_token>
```

Получить токен можно через Auth API:

```bash
curl -X POST https://api.x0tta6bl4.com/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "user", "password": "password"}'
```

## Обработка ошибок

Все API используют стандартные HTTP коды состояния:

- `200` - Успешный запрос
- `400` - Неверный запрос
- `401` - Неавторизован
- `403` - Доступ запрещен
- `404` - Ресурс не найден
- `429` - Слишком много запросов
- `500` - Внутренняя ошибка сервера

Формат ошибки:
```json
{
  "error": {
    "code": "ERROR_CODE",
    "message": "Описание ошибки",
    "details": {},
    "timestamp": "2025-01-01T00:00:00Z"
  }
}
```

## Ограничение скорости (Rate Limiting)

API использует φ-оптимизированные лимиты:

- Бесплатный тариф: 100 запросов/час
- Базовый тариф: 1000 запросов/час
- Профессиональный тариф: 10000 запросов/час
- Корпоративный тариф: Безлимитный

Заголовки ответа содержат информацию о лимитах:
```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 987
X-RateLimit-Reset: 1609459200
```
