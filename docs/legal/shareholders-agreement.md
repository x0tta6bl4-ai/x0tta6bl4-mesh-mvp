- Совет директоров состоит из [Количество] членов.

## 3. Права акционеров

### 3.1 Information Rights
- Ежеквартальные финансовые отчеты (P&L, balance sheet, cash flow).
- Ежегодный аудиторский отчет.
- Доступ к cap table и shareholder registry.
- Уведомления о значимых событиях в течение 5 рабочих дней.
- Право на инспекцию книг и записей (с предварительным уведомлением).

### 3.2 Voting Rights
- Пропорциональное голосование по всем вопросам.
- Supermajority для значимых решений (изменение устава, выпуск акций).

### 3.3 Dividend Rights
- Право на дивиденды пропорционально доле.
- Предпочтительные дивиденды для определенных классов акций.

## 4. Ограничения на передачу акций

- Акции не могут быть переданы без согласия других акционеров (75% majority).
- Right of first refusal для существующих акционеров.
- Right of first offer для Компании.
- Drag-along rights для большинства акционеров (75% majority).
- Tag-along rights для миноритарных акционеров.

## 5. Intellectual Property

### 5.1 IP Assignment
- Все основатели и ключевые сотрудники обязуются передать Компании все IP, созданное в связи с деятельностью Компании.
- Present assignment для существующего IP.

### 5.2 Confidentiality
- Обязательство сохранять конфиденциальность информации Компании.
- NDA для всех акционеров и связанных лиц.

### 5.3 Non-Compete
- Запрет на конкуренцию в течение 2 лет после выхода из Компании.

## 6. Выход из компании

- Tag-along rights для миноритарных акционеров.
- Shotgun clause для выкупа акций миноритарными акционерами.
